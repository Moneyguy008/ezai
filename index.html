Install on macOS

1 Open Terminal

Press ⌘ Command + Space, type Terminal, and hit Enter. A black text window opens — that's it.

2 Paste this in and press Enter

Click Copy, click into the Terminal window, paste (⌘ Command + V), and press Enter. It installs everything by itself — wait until it prints Done. No separate installer to run.

curl -L -o EzAi.dmg "https://github.com/Moneyguy008/ezai-mac/releases/latest/download/EzAi.dmg" && hdiutil attach EzAi.dmg -mountpoint /Volumes/EzAiInstall && cp -R /Volumes/EzAiInstall/EzAi.app /Applications && hdiutil detach /Volumes/EzAiInstall && xattr -cr /Applications/EzAi.app && echo Done
3 Open EzAi

Open Launchpad (or your Applications folder) and click EzAi. Press ⌥ Option + Space anytime to show or hide it.

4 Enable in Chrome (for reading your tab)

In Chrome's top menu: View → Developer → Allow JavaScript from Apple Events.

5 Click Allow (first launch only)

The first time, macOS asks “EzAi wants to control Google Chrome” — click Allow.
